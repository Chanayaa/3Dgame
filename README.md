# 3Dgame
100 Triangle dance

- Create one triangle and move(translates) to all frame window.
- In their position, triangle need move it 28 degrees around their bottom vertex
- Even index triangle rotate left first while odd index triangle move right.
- When rotate left, lock position bottom left to be core for rotated.
- When rotate right, lock position bottom right instead.
